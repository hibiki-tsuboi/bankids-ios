# Bankids (バンキッズ)

**Bank + Kids** の造語から生まれた、子供のお小遣いを管理するiOSアプリです。

## どんなアプリ？

インターネットバンキングとATMを一体化したような、お小遣い管理アプリです。

子供がお年玉やお祝いで大きな金額をもらったとき、全額を渡すと使いすぎてしまうことがあります。Bankidsを使えば、親がお金を預かりながら残高を管理し、子供が必要なときだけお金を渡すことができます。

### 使い方の例

1. 子供がお年玉で1万円をもらったら、アプリに1万円を入金して親がお金を預かる
2. アプリには残高1万円と表示される
3. おもちゃ屋で子供が1,000円使いたいとき、アプリから1,000円を出金して渡す
4. アプリの残高は9,000円になる

## 主な機能

- **入金・出金** - ATMのようにお金の出し入れを記録
- **残高確認** - インターネットバンキングのようにいつでも残高を確認
- **入出金明細** - いつ・いくら出し入れしたかの履歴を確認
- **口座振替** - 親口座から子供の財布口座へお金を振り替え
- **複数口座** - 1人の子供に「親口座」と「財布口座」など複数の口座を管理
- **複数の子供** - 子供ごとにアカウントを切り替えて管理

### 口座の仕組み

```
子供A
├── 親口座      （親が管理するお金）
└── 財布口座    （子供に渡したお金）

子供B
├── 親口座
└── 財布口座
```

親口座から財布口座へ1,000円を振り替えると、親口座は-1,000円、財布口座は+1,000円になります。銀行の口座振替と同じイメージです。
